## Vagrant Laravel

A base Laravel application for use with Vagrant

* Apache2, MySQL 5.5, PHP 5.5, xDebug 2.2
* Default page
  - http://localhost:5472
* Base Migration and Seed file (sample Customer table)
  - http://localhost:5472/customers
* PHP Info Page
  - http://localhost:5472/info
